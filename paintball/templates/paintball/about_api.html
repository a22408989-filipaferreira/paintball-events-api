Paintball Events API

Esta API foi desenvolvida com Django Ninja para gerir eventos de paintball, participantes e equipas.

Ver documentação Swagger Ver Modelo de Dados
🗓️

Entidades

  • Event - representa um evento de paintball
  • Participant - representa uma pessoa inscrita num evento
  • Team - representa uma equipa de participantes
🔗

Relações

  • Um evento pode ter vários participantes.
  • Uma equipa pode ter vários participantes.
  • Um participante pode pertencer a várias equipas.
⚙️

Tecnologias

  • Django
  • Django Ninja
  • Swagger
🏆
