Modelo de Dados

Nesta página é apresentado o modelo de dados da API Paintball Events. O sistema foi construído com três entidades principais e inclui uma relação 1:N e uma relação N:N, conforme solicitado no enunciado.

Entidades

Event

  • name
  • date
  • location
  • price
  • description

Participant

  • name
  • email
  • age
  • event_id

Team

  • name
  • color
  • participants

Relações

Relação 1:N

Um evento pode ter vários participantes, mas cada participante está associado a apenas um evento.

Relação N:N

Uma equipa pode ter vários participantes e um participante pode pertencer a várias equipas.

Resumo

Event 1:N Participant
Participant N:N Team
